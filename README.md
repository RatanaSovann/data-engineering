# Data Engineering

This repo contains the projects related to data engineering.

## Projects

- 🏡 [Data Pipelines: Airbnb + Census Data](https://github.com/RatanaSovann/data-engineering/blob/main/Airbnb%20%2B%20Census%20Data): Built ELT data pipelines using Apache Airflow and dbt Cloud. The pipeline encompasses processing and transforming Airbnb and Census data for Sydney, load it into a data warehouse following a medallion architecture (Bronze, Silver, Gold)and creating a data mart for analytical insights. Ad-hoc analysis is then performed to address key business questions

- 🚗 [Data Processing: NYC Taxi Dataset](https://github.com/RatanaSovann/data-engineering/tree/main/Data%20Processing_%20NYC_Taxi): This project analyses over 1 billion rows of NYC TLC trip data from 2014 to 2024 using Apache Spark on Databricks. It involves large-scale data ingestion, cleaning, and transformation, uses SparkSQL to generate business insights, and builds two machine learning models to predict total fare, evaluated with RMSE.

- [YouTube Trending Data Lakehouse (Snowflake)](https://github.com/RatanaSovann/data-engineering/tree/main/Data%20Lakehouse): Built a data lakehouse pipeline to ingest, transform, and analyse multi-region YouTube data, generating insights on engagement trends using SQL.
